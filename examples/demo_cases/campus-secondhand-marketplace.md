# Campus Second-Hand Marketplace

## Idea

Create a second-hand marketplace for university students, optimized for dormitory delivery, trust, and same-campus exchange.

## Constraints

- Start with one campus only
- Keep acquisition cost very low
- Prioritize matching speed over feature breadth

## Success Metrics

- Number of weekly transactions
- Time from listing to first buyer contact
- Campus penetration rate

## What This Case Should Expose

- Whether density and trust are more important than generic marketplace features
- Whether the local network effect is realistic with the current launch scope
- Whether the business model is strong enough for expansion

## Good Demo Moments

1. Compare direct prompt versus workflow output.
2. Show how finance and marketing differ on monetization timing.
3. Use an intervention to force the plan toward a lower-budget launch.
